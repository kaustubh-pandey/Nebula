# Nebula
A real time search engine to search tweets using Twitter API.
## Requirements
```
Node >= 10.21.0
npm >= 6.14.4
Angular >= 9
```
## To Build and run
```
git clone https://github.com/kaustubh-pandey/Nebula.git
cd Nebula/NebulaUI
npm install
ng build --prod
cd ../NebulaAPI
npm install
npm start
```
Open localhost:3000 in browser
